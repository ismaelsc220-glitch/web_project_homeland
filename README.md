# Tripleten web_project_homeland
 En este proyecto utilice grid para ordenar la sección de "photo" y "place"
 El diseño es responsivo a 800px y a 450px adaptado para cualquier tipo de pantalla.
 Se utilizo la pseudo clase: hover para  el boton de compra y las imagenes, esta función unicamente se activa en escritorio.
 
 Este sitio web tiene como objetivo hacert venta de nfts de ciudades, poniendo una breve información de lo que esta a la venta, utilizando un diseño, sobrio pero a la vez elegante.
